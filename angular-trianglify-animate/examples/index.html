Overview

Angular Trianglify Animate is tiny (2kb) plugin to add support for animate images SVG to your page.

Default options

Option Default Value Type Description
speed 1 number animation speed [s]
easing 'linear' string avaiable: 'in-out-cubic', 'in-out-quintic', 'in-quintic', 'in-quartic', 'in-cubic', 'in-quadratic', 'out-quintic', 'out-quartic', 'out-cubic', 'out-in-cubic', 'out-in-quartic', 'back-in-cubic', 'back-in-quartic', 'out-back-cubic', 'out-back-quartic', 'out-elastic-small', 'out-elastic-big'
vector-x 0 number ***todo***
vector-y 5 number ***todo***
base-max 0 number -
base-min 500 number -
thesa 0.4 number -
